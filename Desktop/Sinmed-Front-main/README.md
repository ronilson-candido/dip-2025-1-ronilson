This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Project Info

This project will run with the following tools:

- React.js
- React Native
- Next.js
- SCSS

Before running on the dev server, you must clone this repo on your folder terminal:

```bash
git clone https://https://github.com/devbragas/Sinmed-Front
```

## Getting Started

First, run the development server (Terminal):

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

This project uses [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to automatically optimize and load Inter, a custom Google Font.
